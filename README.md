**开源语音录制** 一个小型Web应用，用于收集并上传语音片段到云存储，为帮助收集语音数据集用于训练机器学习系统而设计。

基于小型Flask App，利用Javascript在客户端方向进行收集一些特定预置的语音，并Post到服务端。
